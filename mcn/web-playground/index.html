MCN Playground Guide

Quick Start

1. Select an example from dropdown or write your own MCN code

2. Click Run for interpreter mode or CLI Run for real execution

3. Try Serve API to create real REST endpoints from your functions

Button Functions

▶ Run - Execute in playground interpreter

CLI Run - Execute with real MCN CLI (python run_mcn.py run)

Serve API - Start API server, functions become endpoints

Init Project - Create new MCN project with frontend

Generate - Generate API docs, Postman collections, etc.

MCN Syntax Examples

// Variables and Functions var name = "MCN" log "Hello " + name function greet(user) return "Welcome " + user echo greet("Developer")
// Async Tasks task "email" send_email "user@example.com" task "db" query "SELECT * FROM users" var results = await "email" "db"
// AI Integration use "ai_v3" register "my-model" "openai" var response = run "my-model" "Analyze data"

Real API Creation

Write functions in MCN, then click Serve API:

function get_users() return [{"id": 1, "name": "John"}] function create_user(name, email) return {"id": 2, "name": name, "email": email}

These become: GET /get_users and POST /create_user
