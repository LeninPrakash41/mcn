# MCN v3.0 Features Documentation

## 🎉 MCN v3.0 - Natural Language Programming Era

MCN v3.0 represents a major leap forward in AI-powered programming, introducing natural language programming capabilities, advanced AI model management, IoT integration, autonomous agents, and intelligent data pipelines.

## 🚀 New Features Overview

### 1. AI Model Management & Fine-tuning
- **Multi-model Support**: Register and switch between different AI models
- **Custom Model Registration**: Add your own local or cloud models
- **Model Fine-tuning**: Train specialized models on your data
- **Dynamic Model Switching**: Change models based on task requirements

### 2. IoT & Edge Integration
- **Device Registration**: Connect and manage IoT devices
- **Real-time Monitoring**: Read sensor data and device status
- **Command Execution**: Send commands to smart devices
- **Protocol Support**: MQTT, WebSocket, HTTP, CoAP

### 3. Event-driven Programming
- **Event Registration**: Define custom event handlers
- **Asynchronous Processing**: Background event processing
- **Event Triggering**: Programmatically trigger events
- **Workflow Automation**: Chain events for complex workflows

### 4. Autonomous Agents
- **Agent Creation**: Define specialized AI agents
- **Memory Management**: Agents maintain context and memory
- **Tool Integration**: Agents can use MCN functions
- **Collaborative Workflows**: Multi-agent interactions

### 5. AI-powered Data Pipelines
- **Multi-stage Processing**: Chain data transformation steps
- **AI Integration**: Built-in AI classification and extraction
- **Performance Monitoring**: Track pipeline execution
- **Flexible Configuration**: Customize pipeline steps

### 6. Natural Language Programming
- **Code Translation**: Convert English to MCN code
- **Pattern Matching**: Recognize common programming patterns
- **AI Enhancement**: Use AI for complex translations
- **Direct Execution**: Execute natural language commands

## 📖 Usage Examples

### AI Model Management
```mcn
use "ai_v3"

// Register custom models
register("my-gpt4", "openai", {"api_key": "sk-...", "temperature": 0.7})
register("local-llama", "local", {"model_path": "/models/llama-2-7b"})

// Set active model
set_model("my-gpt4")

// Fine-tune a model
train("gpt-3.5-turbo", "sales_data.csv", "sales-predictor")

// Use different models
var summary = run("my-gpt4", "Summarize the quarterly report")
var prediction = run("sales-predictor", "Predict next quarter sales")
```

### IoT Integration
```mcn
use "iot"
use "events"

// Register devices
device("register", "temp_sensor_1", {"type": "temperature_sensor"})
device("register", "smart_light_1", {"type": "smart_light"})

// Set up automation
on event "high_temperature" handle_cooling

function handle_cooling(data) {
    device("command", "smart_light_1", {"command": "dim", "level": 50})
}

// Monitor and trigger
var temp = device("read", "temp_sensor_1")
if temp > 25 {
    trigger("high_temperature", {"temperature": temp})
}
```

### Autonomous Agents
```mcn
use "agents"

// Create specialized agents
agent("create", "data_analyst", {
    "prompt": "You are a data analyst expert",
    "model": "gpt-4",
    "tools": ["query", "ai", "log"]
})

agent("create", "customer_service", {
    "prompt": "You are a helpful customer service agent",
    "model": "gpt-3.5-turbo"
})

// Activate and use agents
agent("activate", "data_analyst")
var analysis = agent("think", "data_analyst", {
    "input": "Analyze Q4 sales: Revenue $2.5M, up 15%"
})
```

### Data Pipelines
```mcn
use "pipeline"

// Create processing pipeline
pipeline("create", "text_processor", [
    {"type": "clean", "params": {"remove_special_chars": true}},
    {"type": "ai_classify", "params": {"categories": ["positive", "negative", "neutral"]}},
    {"type": "ai_extract", "params": {"type": "entities"}}
])

// Process data
var result = pipeline("run", "text_processor", "Great product! Love it!")
```

### Natural Language Programming
```mcn
use "natural"

// Translate natural language to code
var code1 = translate("create a variable name with value John")
// Result: var name = "John"

var code2 = translate("if age is greater than 18 then print adult")
// Result: if age > 18 { echo("adult") }

var code3 = translate("for each user in database do send welcome email")
// Result: for user in database { send_email("welcome", user) }

// Execute natural language directly
translate("create variable greeting with value hello", true)
```

## 🔧 New Syntax Features

### Event Syntax
```mcn
// Register event handler
on event "user_login" handle_login

// Trigger events
trigger "user_login" {"user_id": 123}
```

### Device Operations
```mcn
// Register device
device "register" "sensor1" {"type": "temperature"}

// Read from device
var reading = device "read" "sensor1"

// Send command
device "command" "light1" {"action": "on"}
```

### Agent Operations
```mcn
// Create agent
agent "create" "assistant" {"prompt": "You are helpful"}

// Agent thinking
var response = agent "think" "assistant" {"input": "Hello"}
```

### Pipeline Operations
```mcn
// Create pipeline
pipeline "create" "processor" [{"type": "clean"}]

// Run pipeline
var result = pipeline "run" "processor" "data"
```

### Natural Language
```mcn
// Translate natural language
natural "create variable count with value 0"

// Execute natural language
natural "print hello world" true
```

## 📦 New Packages

### ai_v3 Package
- `register(name, provider, config)` - Register AI model
- `set_model(name)` - Set active model
- `run(model, prompt, options)` - Run AI model
- `train(base_model, dataset, new_name)` - Fine-tune model

### iot Package
- `register(device_id, type, config)` - Register IoT device
- `read(device_id)` - Read device data
- `command(device_id, command, params)` - Send device command

### events Package
- `on(event_name, handler)` - Register event handler
- `trigger(event_name, data)` - Trigger event

### agents Package
- `create(name, config)` - Create autonomous agent
- `activate(name)` - Activate agent
- `think(name, input)` - Agent processing

### pipeline Package
- `create(name, steps)` - Create data pipeline
- `run(name, data)` - Execute pipeline

### natural Package
- `translate(text)` - Translate natural language to MCN
- `execute(text, run)` - Execute natural language directly
